# grade-calculator

## A simple C program that calculates and prints a student's grade based on their numeric score.

## Features

- Takes a student's average score (integer).
- Outputs the corresponding grade:
  - Grade A: for 90–100
  - Grade B: for 80–89
  - Grade C: for 70–79
  - Grade D: for 60–69
  - Grade F: for below 60
